## Projetos

### PDF Intelligence Platform

Projeto principal do portfólio, aplicado em contexto real para transformar documentos operacionais em dados
estruturados e analisáveis.

- Processamento de mais de 500 PDFs.
- Extração, transformação e persistência de dados em PostgreSQL.
- Integração com Power BI para análise.
- Automação de registros antes realizados manualmente.
- Redução de cerca de 2–3 minutos de trabalho manual por registro para execução praticamente instantânea.

### ERP AutoCenter

Sistema interno para autocenter, voltado a organizar processos operacionais e centralizar dados de clientes,
veículos, produtos, estoque e ordens de serviço.

- Desenvolvimento de backend.
- Criação de APIs.
- Banco de dados PostgreSQL.
- Automações para apoio aos fluxos operacionais.
- Integração de processos internos da operação.

**Tecnologias confirmadas:** Python, PostgreSQL, APIs, automações.

### Financial Analytics Lakehouse

Projeto em desenvolvimento para estudo e aplicação de Engenharia de Dados em dados financeiros.

- Integração de dados de fontes como CVM, Yahoo Finance e Banco Central.
- Arquitetura baseada em camadas.
- Modelagem de dados para uso analítico.
- Preparação prática para plataformas modernas de dados.
