## Projetos em destaque

### [PDF Intelligence Platform](https://github.com/devmichaellima/pdf_etl_platform)

**Status:** pipeline funcional aplicado a uma demanda real.

- **Problema:** recibos em PDF exigiam leitura, consolidação e preparação manual dos dados.
- **Solução:** aplicação para upload, extração, tratamento, exportação e persistência dos registros.
- **Fluxo:** PDF → pdfplumber e Regex → Pandas → CSV/Excel ou PostgreSQL → Power BI.
- **Stack:** Python, Streamlit, pdfplumber, Regex, Pandas, SQLAlchemy, PostgreSQL, SQL, Power BI e Plotly.
- **Resultado:** mais de 500 PDFs processados e redução de uma tarefa de 2–3 minutos para poucos segundos.

### [ERP AutoCenter](https://github.com/devmichaellima/erp_autocenter)

**Status:** em desenvolvimento.

- **Problema:** informações de clientes, veículos, serviços e ordens de serviço estavam distribuídas em processos operacionais.
- **Solução:** sistema para estruturar cadastros, regras de negócio, estoque e ordens de serviço.
- **Fluxo:** interface e rotinas Python → serviços e validações → PostgreSQL → consultas operacionais e analíticas.
- **Stack implementada:** Python, PostgreSQL, SQL, HTML, CSS e JavaScript.
- **Resultado atual:** modelagem e rotinas para clientes, veículos, itens e ordens de serviço, dentro de uma operação com cerca de 50 clientes e 70 ordens de serviço por mês.

### [Financial Analytics Lakehouse](https://github.com/devmichaellima/banking_bi)

**Status:** em desenvolvimento.

- **Problema:** dados financeiros e econômicos estão distribuídos entre diferentes fontes públicas.
- **Solução:** pipeline para coletar, organizar e transformar dados de bancos brasileiros e indicadores macroeconômicos.
- **Fluxo atual:** Banco Central, CVM e Yahoo Finance → Python/Pandas → arquivos Parquet → camadas Lake, Bronze e Silver.
- **Stack implementada:** Python, Pandas, Requests, yfinance, PyArrow e Parquet.
- **Resultado atual:** extrações das três fontes e transformação de demonstrações da CVM para análise de quatro bancos.

Outros projetos públicos: [Enriquecimento de Dados com API de CEP](https://github.com/devmichaellima/api_data_enriquecimento) e [SampleScope](https://github.com/devmichaellima/samplescope_streamlit).
