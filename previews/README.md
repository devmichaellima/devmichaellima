# Pré-visualização do README

Para visualizar o perfil antes de publicar, use uma destas opções:

1. Abra o arquivo `README.md` diretamente no GitHub depois de enviar a branch.
2. Crie um Pull Request para revisar a renderização antes do merge.
3. Use a prévia de Markdown do seu editor local, lembrando que alguns serviços externos de badges e estatísticas podem carregar de forma diferente no GitHub.

Antes de publicar, confira:

- se o banner aparece corretamente;
- se os badges carregam;
- se os links de contato funcionam;
- se os projetos sem repositório público não têm links inventados;
- se a leitura está boa em tema claro e escuro.
