# Guia de Conteúdo

Use este guia para manter o perfil profissional, atual e fiel à sua experiência.

## Rotina de Revisão

- Revise o `README.md` pelo menos uma vez por mês.
- Atualize a seção `Atualmente Estudando` conforme sua evolução.
- Remova tecnologias abandonadas ou que não representem mais seu foco.
- Substitua projetos em planejamento por projetos concluídos ou com entregas demonstráveis.
- Adicione métricas comprováveis quando houver novos resultados.
- Evite declarar domínio avançado sem experiência prática suficiente.
- Verifique se LinkedIn, WhatsApp, e-mail, GitHub e repositórios continuam funcionando.

## Como Evoluir o Perfil

Priorize conteúdo que mostre:

- problemas reais resolvidos;
- decisões técnicas tomadas;
- impacto em processos, tempo, custo, qualidade ou tomada de decisão;
- conexão entre negócio, dados, automação, software e inteligência artificial.

## O Que Evitar

- Listas enormes de badges sem contexto.
- Frases vagas como "apaixonado por tecnologia".
- Links para projetos inexistentes.
- Certificações ainda não concluídas apresentadas como concluídas.
- Tecnologias em estudo apresentadas como experiência profissional consolidada.
